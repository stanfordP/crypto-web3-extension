Instructions

  1. Make sure the Crypto Trading Journal Extension is installed and loaded
  2. Ensure the mock API server is running: cd test-ground && npm start
  3. Select your preferred account mode (Demo or Live)
  4. Click "Connect Wallet" to initiate the connection flow
  5. Approve the connection in your wallet (MetaMask, Brave Wallet, etc.)
  6. Once connected, try the various actions (sign message, send transaction, switch network)
  7. Watch the Event Log for provider events (connect, disconnect, accountsChanged, chainChanged)
