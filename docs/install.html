‚ùì Troubleshooting

"Wallet not detected" error

Make sure MetaMask (or another Web3 wallet) is installed and unlocked. The extension requires a wallet to sign authentication messages.

Download MetaMask ‚Üí

"Load unpacked" button not visible

Make sure "Developer mode" is enabled. Look for the toggle in the top-right corner of the extensions page.

Extension disappears after browser restart

Don't delete the extracted folder! The browser loads the extension from that location. If you moved or deleted it, extract the ZIP again and reload.

"Cannot redefine property: ethereum" error

This is a known conflict between wallet extensions (e.g., Phantom and MetaMask). It doesn't affect functionality ‚Äî you can safely ignore this console message.

How do I update the extension?

Download the latest version, extract it to the same folder (overwrite existing files), then go to extensions page and click the refresh icon on the extension card.
